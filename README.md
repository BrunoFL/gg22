## Prerequisites

- node 10+ (14) and npm

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
node ./server.mjs
```

**You need to start client and server together**

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
