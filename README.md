# vue3-socketio-heroku

Automatic deploy on https://vue3-socketio.herokuapp.com/

- node 14


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
